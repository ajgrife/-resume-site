Spectrum Analyzer

A real-time spectrum analyzer with visual output and custom enclosure

Overview

This spectrum analyzer project uses an FFT algorithm on an embedded microcontroller to provide real-time frequency spectrum data. It features a display interface, analog front end, and USB communication.

Technical Details
  • Microcontroller: STM32F4 series
  • ADC Sampling Rate: 48 kHz
  • Display: 2.4" TFT LCD
  • Enclosure: Custom-designed for 3D printing or CNC machining
